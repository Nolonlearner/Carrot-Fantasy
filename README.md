# 萝卜冒险 Carrot Fantasy

## 介绍
这是2023年程序设计范式的期末项目 - 保卫萝卜。

## 特性
- 实现至少3种不同攻击效果和轨迹的防御塔。
- 显示怪物和萝卜的生命值。
- 支持通过击败怪物获得的资源功能，用于购买防御塔。
- 为每个防御塔支持至少2次升级。
- 为萝卜支持至少2次升级。
- 包括至少3种类型的怪物。
- 包括至少2张地图。
- 支持背景音乐。
- 包括关卡选择屏幕和保存进度的功能。
- 添加攻击、建设和怪物死亡的音效，以提高游戏体验。
- 添加特殊技能，如范围伤害、单目标伤害或增益技能，以增加游戏的深度。
- 设计拥有特殊技能或高生命值的boss怪物，以增加游戏的挑战性。

## 使用方法
鼠标点击操作游戏。
